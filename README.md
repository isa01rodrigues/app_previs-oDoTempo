# 🌤️ Weather App - Previsão do Tempo

![HTML Badge](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS Badge](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![JavaScript Badge](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

O **Weather App - Previsão do Tempo** é uma aplicação web moderna e responsiva, desenvolvida com HTML, CSS e JavaScript, com o objetivo de fornecer informações climáticas em tempo real de forma simples, intuitiva e visualmente agradável.

O projeto utiliza a API gratuita da **Open-Meteo**, permitindo a busca de dados meteorológicos sem necessidade de chave de API, tornando a aplicação leve e acessível.

---

## 🚀 Funcionalidades

- Estrutura de páginas desenvolvida com HTML semântico
- Estilização moderna com CSS, utilizando:
  - Variáveis CSS
  - Animações suaves
  - Layout responsivo (desktop, tablet e mobile)
- Integração com API Open-Meteo para:
  - Busca de cidades via geocoding
  - Exibição de dados climáticos em tempo real
- Exibição de:
  - Temperatura atual 🌡️
  - Sensação térmica 🥵
  - Umidade do ar 💧
  - Velocidade do vento 💨
  - Condição climática 🌤️
  - Ícone dinâmico do clima ☁️
- Sistema de busca por cidade
- Tratamento de erros (cidade não encontrada ou falha na API)
- Indicador de carregamento (loading)
- Interface intuitiva e moderna com foco em UX/UI
